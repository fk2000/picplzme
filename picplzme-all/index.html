PicPlzMe!

a simple plugin to show your picplz images anywhere in a pretty lightbox yay!

    How Does It Work

    Simple!

    This is my first plugin for the brand new API from picplz, so for now, there are only a few options.

    • UserName (obviously)
    • Info animation (slide of hover)
    • Lightbox
    • Show Your Info
    • Show Image Data
    • Image Border Color

    The only required parameter is your username. It will default to mine :)

    The animation is how the info is display for the image on hover, either a slide or a fade.

    You can have the images open in a lightbox, powered by the awesome Slimbox2, or have them link to picplx itself.

    Set an option to have your username, avatar and follower count shown

    Show the likes, comments and views count for the images

    Finally, set a color for the border of the image. A lame little add on takes either a hex or a string color.

    Make it go!

    So to set it up simply include the jQuery library, followed by the plugin and the plugin code for slimbox2. You can, of course, combine and minify these for production, I am leaving the parts separate just for showing you how purposes.

    <link rel="stylesheet" href="css/picplzme.css" />
<link rel="stylesheet" href="css/slimbox2.css" />

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.4/jquery.min.js">
<script src="js/jquery.picplzme.js">
<script src="js/slimbox2.js">

    I also included the CSS sheets here, which you would most likely combine and minify, but you get the idea.

    Once the files are included, all you need to go is this;

                $(function() {
                $('#image_list').picplzme({
                    username: 'thatryan'
                });
            });

    Obviously, replace username with your own! This latches onto a ul in the markup with an id of image_list and calls the plugin. Like this,

                <ul id="image_list">

    Options

    Just a few options here,

    username
    This is a string value, your picplz username
    animate
    A string value, either hover or slide. Controls animation for image info
    lightbox
    Boolean, true of false, if you want the image to pop up in lightbox
    userData
    Boolean, true of false, if you want to show your avatar and followers
    imageData
    Boolean, true of false, if you want to show the view,comment and like count
    color
    string value, color name or hex value for a border around images
